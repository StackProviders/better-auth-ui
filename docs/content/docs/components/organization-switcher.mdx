---
title: <OrganizationSwitcher />
description: Beautiful organization switcher component for better-auth organization management.
---

The `<OrganizationSwitcher />` allows users to quickly switch between their organizations, create new ones, and access organization settings.

## Installation

<ShadcnInstall name="org" />

## Usage

```tsx
import { OrganizationSwitcher } from "@/components/organization/organization-switcher";

export default function Nav() {
  return (
    <nav>
      <OrganizationSwitcher />
    </nav>
  );
}
```
